*CSS* -> CASCADE STYLE SHEETS -> FOLHA DE ESTILO EM CASCATA

SELETORES: Informam ao navegador quais elementos html devem ser selecionados para que os estilos sejam aplicados a ele:
//seletor de tag//
p {

}

//seletor de classe//
.cabecalho {

}

//seletor universal//
* {

}

//seletor de descendente//
ul li {

}

PROPRIEDADES: É uma característica (como a cor) cujo valor define o aspecto de como o navegador deve exibir o elemento:
.cabecalho {
	background-color: red;
}

'background-color' = propriedade
'red' = valor

exemplo:
HTML:
<div class="cabecalho">
 Aqui tem um conteúdo qualquer dentro do div
</div>

CSS:
.cabecalho {
 background-color: red;
 }
